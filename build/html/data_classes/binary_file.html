binary_file_2_df(column_names, num_columns, num_rows=-1, size_byte=3, byte_order='big', signed=True)¶

### Purpose: Read a binary file and store the data in a df

#### where: column_names: columns names for the created df num_columns: Number of columns that should be in the new df num_rows: Number of the rows in the new df (preset to -1 so that the value is implicit) size_byte: Number of bytes that should be read at once (This is the number of bytes that make up a value) byte_order: Defines how the bytes of a binary number are arranged in memory. For PFFP binary files the byte_order is “big”
